Rev's Movies Choices of the Week
___________________________________

To view website, open revs_movie.html in your web browser of choice.

To update or change the movies included in the website:
	1. Open entertainment_center.py in a text editor
	2. Follow the comments and change the information for:
		Movie's Title
		Movie's Tagline
		URL or file path for poster image
		Movie's duration
		Movie's MPAA Rating
	3. Save the file.
	4. Open rev_movie_week.py in Idle and run the program.
		A new tab in your web browswer should open with the updated
		web page.
